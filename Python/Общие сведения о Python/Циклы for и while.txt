К выражению while можно пристыковать блок else. И заход в блок else будет произведен при нормальном выходе из цикла while (без применения оператора break).

Также блок else можно пристыковать к циклу for.

Фукнция zip() принимает в себя несколько последовательностей и формирует из них кортежи. Эти кортежи могут возвращаться при итерировании:

list1 = [1, 2, 3]
list2 = [1, 2, 3]

for i, j in zip(list1, list2):
    print(i, j)

1 1
2 2
3 3

Функция zip() обрезает число кортежей до длины минимальной последовательности поступившей на вход.
