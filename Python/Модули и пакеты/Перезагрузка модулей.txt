Код модуля выполняется только один раз на процесс. При импортировании. В том числе весь код выполняется и при
применении оператора from.

Для того, чтобы выполнить повторное выполнение кода импортируемого модуля, необходимо применить функцию reload() модуля importlib.

Пример использования:

----------------------------------
app1:
a = 15

app2:
import time
import app2
from importlib import reload


print(app2.a) -> 15
time.sleep(5) # Тут меняем значение a в файле app2 на 55
reload(app2) # Модуль перезагружается, заново копируется, так сказать
print(app2.a) -> 55
----------------------------------

Из сигнатуры функции видо, что reload() принимает в себя объект типа module. Это, в свою очередь говорит о том, что использование reload(x) в отношении к конструкции from ... import x не возможно, потому что в этом случае в вызывающем коде создается переменная x с типом не module.
