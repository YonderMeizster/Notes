Модули Python - единицы организации программы наивысшего уровня. Представляют собой изолированные пространства имен. Обычно модули соответствуют программам python. Приближенно можно считать, что каждый модуль- отдельная программа.

Как работает импортирование:
Импортиирование можно разделить на три шага:
1) Поиск файла модуля;
2) Компиляция его в байт- код (при необходимости);
3) Выполнение кода модуля для создания объектов, определенных в нем.
Помимо выполнения операторов def для создания объектов функций модуля, также будут исполнены и другие инструкции верхнего уровня.

Оператор import вида import b может распознать:
-	файл исходного кода b.py;
-	файл байт- кода b.pyc;
-	файл оптимизированного байт- кода b.pyo;
-	каталог b;
-	скомпилированный модуль расширения, написанный на C, C++ или другом языке, динамически связываемый при
импортировании (.dll или .so);
-	скомпилированный встроенный модуль, написанный на C и статически связанный с Python;
-	компонент в файле zip, автоматически извлекаемый при импортировании;
-	а также некоторые другие.
