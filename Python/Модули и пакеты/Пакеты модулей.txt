Пакетом модулей зовется каталог с росыпью файликов python.

Примеры использования:
Пусть с файлом вызывающего кода находится каталог folder, внутри которого лежит модуль app2.py
В таком случае импорт этого модуля будет выглядеть следующим образом:

import folder.app2 или from folder.app2 import ... или import folder.app2 as app2

В операторе импортирования запрещено применять синтаксис указания пути, специфичный для платформы:
"C:\Users\Desktop\..." или "/home/user/app/..."

Вот так написать нельзя:

import C:\mycode\dir1\dir2\mod

Вместо этого можно добавить C:\mycode в перменную среды PYTHONPATH или в файл .pth, а в вызывающем коде производить импорт таким образом:

import dir1.dir2.mode
